BILDVISARE
JavaPEG har möjlighet att visa bilder. De bilder som kan visas är sådanna som lagts till i "BILDLISTA"-listan i fliken "SÖK / VISA BILDER". Se hjälpavsnitt för "VISA BILDER" för hjälp med att skapa en bildlista.

Om en lista med bilder skapats enligt ovan så kan knappen "Visa valda bilder i bildvisaren" i tabben "SÖK / VISA BILDER" tryckas på för att få fram bildvisaren, annars skapa en bildlista först.

Egenskaper för bildvisaren:

Bildvisaren innehåller tre huvudelement: En lista med tillgängliga bilder för bildvisaren att visa (visas som en lista med tumnagelversioner av bilderna.) som finns till vänster i användargränssnittet. I mitten finns själva ytan för att visa upp bilderna på. Till höger finns en panel som visar detaljerad metadatainformation om visad bild.

Tumnagellistan till vänster och metadatapanelen till höger går att dölja för att ge mer plats för att visa bilder.

Det finns stöd för följande aktiviteter i bildvisaren:

  • Navigering framåt och bakåt i listan med bilder, med antingen knapp i knapplistan, eller genom högerklicksmeny på själva bilden, eller med snabbkommando eller med höger eller vänster piltangent.
  • Navigation till valfri bild i listan genom att markera en bild i tumnagelöversikten till vänster.
  • Anpassa storleken på visad bild så att den får plats i tillgängligt utrymme för bildvisarytan.
  • Automatiskt anpassa storleken på visad bild så att den får plats i tillgängligt utrymme för bildvisarytan när en bild väljs.
  • Automatiskt rotera visad bild enligt Exif information.
  • Rotera visad bild 90 grader medurs.
  • Rotera visad bild 90 grader moturs.
  • Använda dag och släpp for att flytta på bilden.
  • Klicka i en mindre version av den visade bilden, för att centrera den visade bilden vid den klickade punkten.
  • Förstora eller förminska den visade bilden med mushjulet.
  • Det är möjligt att centrera bilden med ett knapptryck.
