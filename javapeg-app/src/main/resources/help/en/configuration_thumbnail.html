Thumbnail Creation:

If embedded thumbnail is missing or corrupt, create a temporary:
If an image is missing an thumbnail or if it is corrupt then a temporary thumbnail can be created by JavaPEG to be displayed in the places where thumbnails exists.

Width of the created thumbnail:
Specifies in pixels, how wide the created thumbnail shall be.

Height of the created thumbnail:
Specifies in pixels, how high the created thumbnail shall be.

Thumbnail Creation Algorithm:
Specifies which algorithm to use when a thumbnail is created. There are two possible algortims to choose between: SMOOTH and FAST. The first one is slower but provides a nicer looking result.


Thumbnail Cache:

Enable Thumbnail Cache:
Activation of thumbnail cache will result in a more responsive application since the thumbnails from images which have already been displayed once in JavaPEG will be stored in the memory of the computer and this means that if the same thumbnail shall be displayed once more, then the thumbnail can be fetched from the computer memory instead from the source image, which is much faster.

Thumbnail cache size:
Indicates how many thumbnails that currently is stored in the thumbnail cache.

Thumbnail cache size max:
Specifies the maximum amount of thumbnails that can be stored in the memory based thumbnail cache.

Clean Thumbnail chache:
Empties the thumbnail cache from all currently stored thumbnails.


Thumbnail Tooltips:

Disabled:
No tooltips will be displayed when the mouse pointer is over thumbnails.

Enabled:
Tooltips will be displayed when the mouse pointer is over an thumbnail.

Extended:
A tooltip with more information will be displayed, compared to the "Enabled" setting, when the mouse pointer is over an thumbnail.
