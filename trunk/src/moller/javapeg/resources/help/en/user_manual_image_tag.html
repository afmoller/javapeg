TAG IMAGES
It is possible to tag images by assign searchable values to them.

  1. Select the tab TAG IMAGES.
  2. Select a directory with images that shall be tagged in the tree structure to the left.
  3. Tag an image by clicking any of the images in the loaded image overview which was loaded in step 2 above. Enter a comment of unrestricted length in the field with the headline COMMENT. It is also possible to assign an image a classification on a scale from 1 to 5. Besides from this it is also possible to assign an unlimited amount of categories to an image.
